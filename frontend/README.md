# Frontend - Vue 2 + Vuetify

Interface frontend do projeto ADMF01-2025.2

## Requisitos

- Node.js 14+
- Yarn

## Instalação

```bash
yarn install
```

## Desenvolvimento

```bash
yarn serve
```

## Build

```bash
yarn build
```

## Estrutura

- `src/` - Código fonte
- `public/` - Arquivos estáticos
- `tests/` - Testes
